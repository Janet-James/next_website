LAND, BUILDING & UTILITY INFORMATION

Identification Tool enables the user to know the tool specific Attributes/details on the location where the user clicked. City Profiling provides three types of feature identifications:

  • Land identification – to get details of land if land details available at the clicked location

  • Building identification – to get details of building if building details available at the clicked location

  • Utility identification - to get details of utilities if utilities detail available at the clicked location

  • By clicking on map to get information of clicked point of map and show information with layer based and user based constraints.

  • Raise or request about issues to respective department

  • Bill payment request if bill not paid

  • Making new request for any connections or permission request

LAND & BUILDING SEARCH

The Search allows to find for a particular Land or Building and fetch their details on the map. It has auto-complete feature which show the possible matches based on the user entry which gives an insight to the user about the available features.

Search by land:

User can search layer based on the land. It shows the information with layer based and user based constraints. It enables user to raise or request about issues to respective department.

Search by Building:

User can search layer based on the building. It shows the information with layer based and user based constraints. It enables user to raise or request about issues to respective department.

BUILDING PERMIT

Building Permit is formal permission to begin the construction, demolition, addition or renovation on your property. As part of the Building Permit process, Issuing Authority must review your plans to ensure they comply with the National Building Code, local Zoning By-laws and other Applicable Law.


The city requires permits to ensure safe building, effective land use, effective water supply, Electricity planning and high-quality site planning. Building regulation enforcement for new building. Existing building permits to ensure change land use, building alteration and building demolishing. It shows utility services details and send service request.



New Building

New building permit can be raised here. Check Building Permit Request link below the Land information Tab in the Details Panel for the New Building.


Existing Building

By selecting the land parcel number, the existing building details will be fetched. It enables to update the details of building area, building purpose, building height, floor count, construction type, utility services and fees. The utility services include electricity permit, water permit, and sewer permit. There is an option to upload the building plan.


SERVICE REQUEST

Service Request can be raised for basic utilities – water, electricity, sewerage, telecom etc. User can view the available utility services, raise new service request, log service problems.

Request for water services

Water Service Request can be used by citizens to report issues with water, sewer, and storm water service or infrastructure. This function allows the public to complete the service request form and submit the location of the issue. New service requests can then be used by a utility to investigate the situation and create any necessary work orders.


Request for electricity services

BUILDING VIOLATIONS

Building code that has rules for the conditions in buildings. Building Violations is recorded in the system and shown. Building violation, when a hazardous or non-hazardous condition occurs in a real estate property, without a permit, issued by a municipal agency, such as a Department of Buildings.

  • Building code that has rules for the conditions in buildings.

  • Building Violations is recorded in the NCPS and displayed on the map.

  • See the Violations Details when hovered/clicked on building violations.

  • Filter the violations based on suburb & type

  • This option enables to find the violations based on suburb.

  • Find zone/area which is not matched with building violations

  • Generate violations report

INCIDENT REPORTING

The purpose of the incident report is to find the exact details of the occurrence of the utility where the issue happened. This information may be useful in the future when dealing with liability issues stemming from the incident. Combining these incidents and mapping out their geographic attributes allows one to visualize the spatial visualization of the incidents happened.

  • Raise incident - To log the incident and sent to the respective stockholder for some action/decision making

  • The issues related to the utilies are captured with images and statistics report will be generated based on that.

  • Incident Status - It shows the distribution of incidents which has been raised. Location of incident is represented as a marker in map.



BI & SPATIAL REPORTS

This functionality helps to analyze and identify about the area/zone. It is helpful to specific department to analyze about the zone like that. The area/zone which is recently developed, the zone which concentrate more to develop, etc.,

  • Revenue Reports - Revenue based reports

  • Violation Reports – Violation based reports
